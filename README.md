# Kiko AI Study Planner 📚✨

## Overview

Kiko AI Study Planner is a study planning web application designed to help students organize tasks, manage study schedules, and improve productivity. The application provides intelligent task organization, progress tracking, and personalized study assistance through a simple and user-friendly interface.

## Features

* ✅ Add and manage study tasks
* ✅ Organize tasks based on priority
* ✅ Track task completion and progress
* ✅ Personalized study assistance
* ✅ Responsive design for desktop and mobile devices
* ✅ Simple and intuitive user interface
* ✅ Local data storage

## Technologies Used

* HTML5
* CSS3
* JavaScript


## Installation

1. Clone the repository:

```bash
git clone https://github.com/anushreeanil7/Kiko-AI-Study-Planner.git
```

2. Open the project folder in VS Code.

3. Launch the project using Live Server or open `index.html` in a browser.
